 Image Steganography Project

## 📌 Description
This project is used to hide secret messages inside an image using the LSB (Least Significant Bit) technique and retrieve them securely.

## 🚀 Features
- Encode (hide) secret message into an image
- Decode (extract) hidden message from image
- Simple and easy implementation using Python

## 🛠️ Technologies Used
- Python
- Pillow (PIL)

## ▶️ How to Run

1. Install required library:
   pip install pillow

2. Run the program:
   python main.py

3. Choose option:
   - 1 → Encode message
   - 2 → Decode message
